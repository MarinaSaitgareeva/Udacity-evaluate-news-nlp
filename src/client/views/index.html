What is Sentiment Analysis?

Sentiment Analysis is solution for performing a detailed multilingual sentiment analysis of texts from different sources.

The text provided is analyzed to determine if it expresses a positive/negative/neutral sentiment.

Besides polarity at sentence and global level, Sentiment Analysis uses advanced natural language processing techniques to also detect the polarity associated to both entities and concepts in the text.

Additionally, Sentiment Analysis detects if the text processed is subjective or objective and if it contains irony marks [beta] at a global level, giving the user additional information about the reliability of the polarity obtained from the sentiment analysis.
